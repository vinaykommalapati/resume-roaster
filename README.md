# AI Resume Roaster 🔥🎤

A brutally honest AI that **roasts your resume like a comedian** — then gives you real tips to fix it.

Built by [Vinay Kommalapati](https://github.com/vinaykommalapati) — AI Engineer & Full Stack Developer.

---

## What It Does

1. You paste your resume text
2. The AI **roasts it savagely** 😂
3. Then gives you **5 real fixes** to make it better

## Setup & Run

### 1. Clone the repo
```bash
git clone https://github.com/vinaykommalapati/resume-roaster.git
cd resume-roaster
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Get your free Groq API key
- Go to [console.groq.com](https://console.groq.com)
- Sign up free and copy your API key

### 4. Set your API key
```bash
# Windows
set GROQ_API_KEY=your_api_key_here

# Mac/Linux
export GROQ_API_KEY=your_api_key_here
```

### 5. Run it
```bash
python roaster.py
```
